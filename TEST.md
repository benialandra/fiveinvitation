# Dokumentasi Testing

File ini disediakan untuk keperluan mencatat instruksi pengujian (testing) secara manual maupun otomatis pada proyek FiveInvitation.

Untuk menjalankan linter dan static checks, Anda bisa mengeksekusi:
```bash
npm run lint
```

*(Anda dapat mengisi panduan testing End-to-End di sini pada masa depan).*
